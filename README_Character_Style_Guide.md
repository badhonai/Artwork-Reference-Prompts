# Character Identity + Style Prompt Guide

Generate brand-new artwork while keeping a character's identity
recognizable.

## How it Works

Upload **two reference images**:

1.  **Identity Reference** (Required)
    -   The character or mascot you want to keep consistent.
    -   Used only for facial identity.
2.  **Style Reference** (Optional but Recommended)
    -   Upload one of your favorite artworks from **Spotlight**.
    -   The model should borrow only the **art style** (line work,
        coloring, rendering, lighting, brushwork, etc.), **not** the
        composition or characters.

> The identity reference defines **who** the character is.
>
> The style reference defines **how** the artwork should look.

------------------------------------------------------------------------

# Identity Prompt

``` text
Use the uploaded image ONLY as an identity reference. Do NOT copy, trace, recreate, or imitate its composition, pose, clothing, background, camera angle, lighting, or objects.

Identity Preservation:
• Keep the character instantly recognizable by preserving the face shape, hairstyle silhouette, eye design, eyebrow style, mouth shape, facial proportions, and overall facial identity as closely as possible.
• Keep the mascot recognizable by preserving only its face, eye shape, mouth, and overall personality.
• Redesign everything else from scratch, including pose, clothing, accessories, body proportions, environment, composition, perspective, and storytelling.
• The result must be a completely new illustration, never a remake of the reference.
```

------------------------------------------------------------------------

# Style Prompt

``` text
Match the rendering style of the uploaded style reference.

Art Style:
• Clean anime illustration with bold, smooth black outlines.
• Bright, vibrant flat colors using the same palette philosophy and coloring approach as the style reference.
• Minimal cel shading with crisp shadow shapes.
• Soft, subtle highlights where appropriate.
• Smooth color fills with no painterly texture.
• High-contrast, polished digital artwork.
• Consistent line weight and professional anime-quality finish.
• Expressive facial expressions and dynamic poses.
• Appealing stylized anatomy and strong silhouettes.
• Sharp edges and clean color separation.
• High-resolution, studio-quality illustration.

Creative Rule:
• Borrow only the visual style.
• Do NOT copy the pose, composition, camera angle, background, props, clothing, or scene from the style reference.
• Produce a completely original illustration.
```

------------------------------------------------------------------------

# Best Practice

-   Identity image → keeps the same character.
-   Spotlight artwork → supplies the visual style.
-   Describe a brand-new scene in your prompt.
-   Never ask the model to recreate the reference image.

## Example

**Identity Image** - Your original character

**Style Image** - Your favorite Spotlight artwork

**Scene Prompt** \> Surfing through a giant tropical wave at sunset with
a smiling blue mascot, dynamic action pose, cinematic perspective.

The output should have: - The same recognizable character. - The visual
style of the Spotlight artwork. - A completely original composition and
scene.
